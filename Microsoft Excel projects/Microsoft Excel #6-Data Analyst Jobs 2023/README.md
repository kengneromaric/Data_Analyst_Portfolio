# Microsoft Excel #6-Data Analyst Jobs 2023

Bloom is a recruiting agency.

They had collected data for job applications from the opportunities listed on their website.

To optimize their application process, they wished to review application data for data analytics jobs for the year 2023.

However, they were overwhelmed by the number of applications.

They requested my help for this analysis.

They wished to get a summary of the application data so they could get the exact information they needed.

I cleaned the data and analyzed it.

I created a summary table containing the key insights they requested.

-The total number of applications received per month

-Total applications received for the roles during that year

-The average number of applications received per month

-Which months had the lowest and highest amount of applications
